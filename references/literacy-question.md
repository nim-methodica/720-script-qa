# שאלה אוריינית — הגדרה ודרישה

## ההגדרה (4 שלבים)

שאלה אוריינית בודקת מעבר מסיטואציה מהעולם האמיתי לפתרון מתמטי וחזרה. ארבעת השלבים:

1. **פירוש והבנה של הסיטואציה** — הבנת ההקשר מהעולם האמיתי.
2. **מידול** — בדרך כלל העברת הסיטואציה לייצוג מתמטי.
3. **פתרון מתמטי** — ביצוע החישוב/הפעולה.
4. **פרשנות או תיקוף** — מה זה אומר ביומיום (פרשנות), או: אם נשנה את הסיטואציה — איך זה ישפיע על
   המודל שבנינו (תיקוף).

## הדרגתיות

- **לא חובה שכל 4 השלבים יופיעו במפורש בשאלה אחת.** אפשר בהדרגה: שלב 2, אחר כך 3, אחר כך 4.
- **אם מופיע רק שלב 4** — המשמעות היא שהילד צריך לעבור בראש את כל השלבים שקדמו לו (1→2→3→4).
- שאלה שהיא רק "פתרון מתמטי" (שלב 3 בלבד) ללא הקשר/מידול/פרשנות — **אינה** אוריינית.

## הדרישה

**≥1 שאלה אוריינית בכל רכיב** (רכיבים 1, 2, 4 — רכיב 3 אינו באחריותנו).

## איך לזהות בתסריט

- חפש שאלה שמתחילה מהקשר מהעולם האמיתי (סיפור/תרשים/מדידה) ודורשת מידול → חישוב → פרשנות/תיקוף.
- **דוגמת-חיוב (תסריט "קנה מידה", שקפים 46–50):** "תרשים חדר ילדים, שולחן 7 ס"מ בתרשים / 1.4 מ'
  במציאות → מהו קנה המידה?" — פירוש (חדר), מידול (יחס), פתרון (140:7 → 20:1), והמשך לתיקוף
  (פריסת שטיח לפי קנה המידה). זו שאלה אוריינית תקינה.
- אם ברכיב כלשהו אין אף שאלה כזו — דגל **חוסם** ברמת-הרכיב, וציין באיזה רכיב חסר.
