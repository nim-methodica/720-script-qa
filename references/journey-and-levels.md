# מסע-הלמידה, רמות-תפקיד וטקסונומיה

מקור: מייל אפרת פרידמן + תמלול חפיפה + טקסונומיית-מתמטיקה (אמ"ה, מבוסס בלום/אביטל-שטולוורס).

## מקטעי מסע-הלמידה (#28)

כל מקטע צריך להיות נוכח ובמקטע נפרד:
**HOOK → קישור לנושא → הקניה → תרגול מונחה (מתמטיקה בלבד) → 2 שאלות חימום → תרגול סטנדרטי →
תרגול בסיסי → סטנדרטי ב → משימת כיתה → תרגול מתקדם → פריט העשרה (לא חובה) → 2 שאלות שיא.**

- "תרגול מונחה" קיים **רק במתמטיקה**.
- "פריט העשרה" לא חובה (לרוב בין מתקדם לשיא).
- ההסמכה מתרחשת רק אחרי שאלת השיא (הפריט האחרון).
- **מבנה-רכיבים עדכני (5–6 רכיבים):** כל **שאלת שיא היא רכיב עצמאי** עם שקף-חוצץ+ID משלה. יחידה
  עם 2 שאלות שיא = 6 רכיבים (1 מבוא-ובסיס · 2 תיקון-אדפטיבי · 3 משימת-כיתה · 4 מתקדם · 5 שיא-א ·
  6 שיא-ב). שאלות-שיא שארוזות יחד או בתוך רכיב "מתקדם" = ממצא #8/#28 (יש לפצל).

## רמת-תפקיד וטקסונומיה (#29)

**תיוג רמה = השלב** (חימום / סטנדרטי / בסיסי / סטנדרטי-ב / מתקדם / שיא). כל שאלה צריכה להיות
מתויגת לשלב, **והדרישה הקוגניטיבית שלה צריכה להתאים לרמה הטקסונומית הצפויה לאותו שלב** (אימות
סופי = מומחה תוכן).

### טקסונומיית מתמטיקה (אמ"ה — 4 רמות)
1. **ידע וזיהוי** — שחזור/זיהוי מושגים, עובדות, הגדרות, נוסחאות.
2. **חשיבה אלגוריתמית** — חישוב לפי אלגוריתם שגרתי (זהה→בחר→יישם; לינארי).
3. **חשיבה תהליכית** — קישור מושגים, התאמת מודל לסיטואציה מילולית, פתרון מתוך תובנה (≈ הבנה+יישום).
4. **חיפוש פתוח והנמקה** — ניסוח-מחדש, ניתוח, סינתזה, חקר, הנמקה; לא-לינארי (≈ אנליזה+סינתזה).

### טקסונומיית מדעים (12 פעלים, ב-3 דרגות)
- **דרגה 1 — זיהוי/ידע:** זיהוי (Identifying) · תיאור (Describing) · איתור מידע (Retrieving
  Information) · מתן דוגמאות (Providing Examples).
- **דרגה 2 — הבנה/יישום:** קישור (Making Connections) · פירוש (Interpreting) · שימוש במודל או
  בפרוצדורה (Applying a Model or Procedure) · הסבר (Explaining).
- **דרגה 3 — חשיבה מסדר גבוה:** הנמקה מדעית (Providing Scientific Reasoning) · ניתוח (Analyzing) ·
  סינתזה (Synthesizing) · הערכה והצדקה (Evaluating and Justifying).

### טבלת שלב → רמה צפויה
| שלב | מדעים (דרגה/פעלים) | מתמטיקה (אמ"ה) |
|---|---|---|
| משולב בהקניה | דרגה 1: זיהוי / איתור מידע / מתן דוגמאות | חשיבה אלגוריתמית |
| חימום | דרגה 1: זיהוי / תיאור — **וקל** (חוויית הצלחה) | ידע וזיהוי |
| סטנדרטי (כולם) | דרגה 2: קישור / פירוש / שימוש במודל או בפרוצדורה / הסבר | חשיבה אלגוריתמית |
| בסיסי | דרגה 2: קישור / פירוש / מודל / הסבר | חשיבה אלגוריתמית |
| סטנדרטי ב | דרגה 1: זיהוי / איתור מידע / מתן דוגמאות | חשיבה אלגוריתמית |
| מתקדם + שיא | דרגה 3: הנמקה מדעית / ניתוח / סינתזה / הערכה והצדקה | חשיבה תהליכית + חיפוש פתוח והנמקה |

## דרישות-תפקיד פר-שלב (#30)

- **משוב מתפתח:** ≥1 שאלה בכל אחד מ: סטנדרטי / בסיסי / מתקדם.
- **ריבוי נקודות מבט:** ≥1 בסטנדרטי.
- **רב-תחומית:** ≥1 במתקדם.
- **גיוון סוגי-שאלה:** חד/רב-ברירה, השלמה ממחסן, הוט-ספוט.
- **תזמון רמז:** סטנדרטי = רמז **אחרי** טעות ראשונה; בסיסי = רמז **עם** הצגת השאלה + רמז מפורט
  יותר (שאלה-למחשבה; שקלו אודיו+תמלול).
- **חימום = קל** — שאלות זכירה קלות שמייצרות חוויית הצלחה (לא קשות/מורכבות).

## אמיל"י — הבהרת הצורך הלימודי ללומד (#31)

היחידה מבהירה ללומד **למה** הוא לומד זאת / איזה צורך זה משרת — לא צוללת לתוכן בלי מסגור.
- בפתיחה: ההוק → "קישור לנושא" מבהירים את הרלוונטיות/הצורך.
- במשוב המתפתח: מסביר ללומד מה קורה עכשיו ונותן ודאות (מסגר את הצורך).
- דגל אם היחידה מלמדת תוכן בלי שהובהר ללומד הצורך/הרלוונטיות.
