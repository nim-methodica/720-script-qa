# script-qa — בקרת איכות לתסריטי הדרכה (Claude Code Skill)

סקיל ל-Claude Code שמבצע **בקרת איכות אוטומטית** על תסריטי הדרכה שלמים (PowerPoint, פרויקט 720
מדעים/מתמטיקה לכיתות ז'–ח'), ומפיק **דוח מפורט לפי שקף** עם ממצאים והמלצות תיקון.

> **הסקיל לא מתקן** — הוא מדווח בלבד. ההחלטה אם לתקן היא של מפתח/ת ההדרכה.

## מה הוא בודק

35 בדיקות ב-7 קבוצות (פירוט מלא: [`references/qa-dimensions.md`](references/qa-dimensions.md)):

1. **שפה ופורמט** — עברית בגובה-עיניים, רווחים כפולים, כתיבה א-מגדרית, מתמטיקה-טיפוגרפיה.
2. **נכונות תוכן** — נכונות מדעית/מתמטית (מול מקור-סמכות), משמעת היקף, טרמינולוגיה.
3. **מבנה ומזהים** — 5–6 רכיבים (כל שאלת שיא = רכיב) + חוצצים + ID, רציפות מזהים, שלמות טבלת-מטא, הערות הפקה.
4. **תקינות שאלות** — עקביות בין-מצבים, שלמות מצבים, הצבעת תשובה↔סימון↔משוב, פיזור-תשובות, רמז, טון, מסיחים.
5. **פדגוגיה ומעורבות** — הוק, התאמה לגיל, ≥1 יישומון, ≥1 שאלה אוריינית/רכיב, סגירת-לולאה, עומס-קריאה, "הצג אל תספר".
6. **מסע-למידה, רמות וטקסונומיה** — שלמות מסע-הלמידה, תיוג-שלב+התאמה טקסונומית, דרישות-תפקיד פר-שלב, אמיל"י.
7. **אותנטיות, רלוונטיות ויישור-ליעד** — נושא אמיתי (לא מומצא/מופרך), רלוונטיות לגיל ז-ח, יישור כל שאלה ליעד, ורכישת המיומנות בפועל בסוף היחידה מול היעד.

## התקנה

העתיקו את התיקייה הזו ל-`.claude/skills/script-qa/` בתיקיית-העבודה של הפרויקט:
```
<פרויקט>/.claude/skills/script-qa/
```
ב-Claude Code הסקיל יזוהה אוטומטית. הפעלה: בקשו "בדוק/תקף את התסריט" או `/script-qa`.

## דרישות

- **markitdown** (חילוץ טקסט מ-PPTX): `pip install "markitdown[pptx]"`
- **Node.js** (המרת הדוח ל-HTML): `scripts/md-to-html.js` (ללא תלויות חיצוניות).
- **Python 3** (הסורק הדטרמיניסטי): `scripts/scan.py` (ללא תלויות).

## תהליך (תקציר)

1. חילוץ טקסט מה-PPTX (`markitdown`, UTF-8).
2. סריקה דטרמיניסטית מלאה (`scripts/scan.py`) — רווחים, טיפוגרפיה, מזהים, עומס-מילים.
3. ניתוח מבנה (רכיבים, מקטעי-מסע, מצבי-שאלה).
4. מעבר בקרה שיפוטי על **כל** שקף (תסריט גדול → סוכני-משנה מקבילים).
5. הפקת `report.md` + `report.html` בתיקיית `reports/`.

פירוט מלא: [`SKILL.md`](SKILL.md).

## מבנה החבילה

```
SKILL.md                  — הנחיות התהליך מקצה-לקצה
references/                — קטלוג הבדיקות + מקורות-סמכות + טקסונומיות + תבנית-דוח
scripts/scan.py            — סורק דטרמיניסטי (read-only)
scripts/md-to-html.js      — ממיר דוח MD→HTML (RTL, צבעי-חומרה)
examples/                  — דוח-דוגמה
```

## מקורות-סמכות

בדיקות הנכונות (#5/#6/#7) מצליבות מול ספרי-לימוד וטקסונומיות (משרד החינוך). הקבצים עצמם
**אינם נכללים בריפו** (זכויות-יוצרים) — יש להניחם מקומית תחת `.claude/input/`. ראה
[`references/authority-sources.md`](references/authority-sources.md).
