התקנת הסקיל 720-script-qa

בקרת איכות לתסריטי הדרכה 720 · מדריך התקנה למחשב חדש

בקצרה: פתיחת ה-zip למקום הנכון היא רק שלב אחד. צריך גם להתקין Python + שתי חבילות pip, ורצוי Node.js. בלי אלה הסקיל לא ירוץ. הפירוט מטה.

1 פתיחת החבילה למיקום הנכון

פתחו את 720-script-qa.zip כך שהתוכן יישב ישירות תחת תיקיית הסקילים של המשתמש:

C:\Users\<שם-המשתמש>\.claude\skills\720-script-qa\

SKILL.md חייב להיות ישירות בתוך התיקייה הזו — לא בתוך תת-תיקייה עוטפת. Claude Code מזהה סקילים לפי המבנה הזה; אם נוצרה תיקייה מיותרת באמצע, הסקיל לא יזוהה.

איך בודקים שהמבנה תקין

אחרי הפתיחה, בתוך ...\skills\720-script-qa\ אמורים להופיע:

  • SKILL.md, README.md, INSTALL.html, CHANGELOG.md
  • התיקיות scripts\ , references\ , examples\ , tests\

2 התקנת התלויות (במחשב — לא בתוך ה-zip)

רכיבפקודה / מקורבשביל מהנדרש?
Python 3 python.org (לסמן «Add to PATH») מריץ את כל הסקריפטים חובה
markitdown python -m pip install "markitdown[pptx]" חילוץ הטקסט מה-PPTX (שלב ראשון) חובה
python-pptx python -m pip install python-pptx geometry_scan.py — מיקום סמני-✓, גלישות, חילוץ תמונות חובה
Node.js nodejs.org (LTS) הפקת דוח ה-HTML (md-to-html.js) רצוי

שורת התקנה אחת (העתקה-הדבקה לטרמינל)

python -m pip install "markitdown[pptx]" python-pptx
שימו לב: אם python לא מזוהה, נסו py -m pip install ... (Windows) או python3 -m pip install .... בלי Node.js הסקיל עדיין עובד — פשוט מפיק רק report.md בלי גרסת ה-HTML.

3 קובצי מקורות-הסמכות אופציונלי

בדיקות הנכונות (#5/#6/#7) מצליבות מול ספרי-לימוד וטקסונומיות. הקבצים האלה אינם נכללים בחבילה (זכויות יוצרים). כדי להפעיל הצלבה מול מקור, הניחו אותם מקומית תחת:

<פרויקט>\.claude\input\

בלעדיהם הסקיל עדיין רץ — בדיקות הנכונות פשוט יסומנו «לאמת מול מומחה תוכן» במקום הצלבה אוטומטית. פירוט: references/authority-sources.md.

4 הערת עברית ב-Windows

הסקיל מריץ את markitdown עם PYTHONUTF8=1 באופן פנימי כדי שהעברית לא תצא כ-?. בדרך כלל אין צורך בפעולה — רק ודאו שה-Python מותקן תקין וש-pip עבד.

5 בדיקת שהתקנה עבדה

  • פתחו את Claude Code בפרויקט כלשהו והריצו /720-script-qa — הסקיל אמור להופיע.
  • בדיקה מהירה של התלויות מהטרמינל:
python -c "import markitdown, pptx; print('python deps OK')"
node --version   # אמור להדפיס מספר גרסה
אם שתי השורות עברו בלי שגיאה — הסקיל מוכן לשימוש: העלו תסריט PPTX ובקשו «בצע בקרה מקיפה».
לגבי עדכונים: החבילה שנפתחה היא צילום-מצב (בלי git). כדי לעדכן בעתיד — פותחים zip חדש לאותה תיקייה (דורס), או מושכים git pull מהמאגר github.com/nim-methodica/720-script-qa. אין עדכון אוטומטי.
